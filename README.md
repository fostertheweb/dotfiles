## Setup

```bash
git clone git@github.com:fostertheweb/dotfiles.git ~/.dotfiles
```

```bash
cd ~/.dotfiles
```

```bash
./install.js
```

### Requirements

- [Bun](https://bun.sh) - Install via Homebrew: `brew install bun`
- [Stow](https://www.gnu.org/software/stow/) - Install via Homebrew: `brew install stow`
