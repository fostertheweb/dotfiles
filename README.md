## Setup

> You will need the command-line tool `stow` installed on your system.

```bash
git clone git@github.com:fostertheweb/dotfiles.git ~/.dotfiles
```

```bash
cd ~/.dotfiles
```

```bash
stow tmux zsh neovim tig
```
